# Features Overview

Quak combines a traditional spreadsheet grid with SQL-powered data capabilities. Here's everything it can do.

## At a Glance

| Feature | Description |
|---------|-------------|
| [AI Chat Assistant](/features/ai-chat) | Natural-language spreadsheet management with LLM tool calling |
| [Multiple Views](/features/views) | Grid, Kanban, Calendar, and Gallery views over the same data |
| [Cell Types](/features/cell-types) | 7 types — text, number, checkbox, dropdown, date, formula, markdown |
| [SQL Queries](/features/sql-queries) | In-browser SQL via DuckDB-WASM with syntax highlighting |
| [Charts](/features/charts) | Bar, line, and pie charts from query results |
| [Import & Export](/features/import-export) | Drag-and-drop CSV/TSV/JSON import, CSV/JSON export |
| [Search & Filtering](/features/search-filtering) | Full-text search, per-column floating filters, multi-column sort |
| [Undo & Redo](/features/undo-redo) | 50-action stack covering cell edits, rows, and columns |
| [Sheet Management](/features/sheet-management) | Multiple sheets, templates, column management |
| [Keyboard Shortcuts](/features/keyboard-shortcuts) | Quick access to all major operations |
| [Dark Mode](/features/dark-mode) | Light, dark, and system theme with smooth transitions |
| [Responsive Design](/features/responsive-design) | Mobile-first layout with bottom navigation |
