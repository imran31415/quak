# Keyboard Shortcuts

Press **?** at any time to open the keyboard shortcuts palette.

## Available Shortcuts

| Shortcut | Action |
|----------|--------|
| `?` | Show keyboard shortcuts palette |
| `Ctrl+Z` | Undo last action |
| `Ctrl+Y` | Redo last action |
| `Ctrl+Shift+Z` | Redo last action (alternative) |
| `Ctrl+F` | Open search bar |
| `Ctrl+Enter` | Run SQL query (when query panel is open) |
| `Escape` | Close dialogs, search bar, or query panel |

::: tip Mac Users
Use `Cmd` instead of `Ctrl` for all shortcuts on macOS.
:::

## Grid Navigation

Standard AG Grid keyboard navigation also applies:

| Key | Action |
|-----|--------|
| Arrow keys | Move between cells |
| `Enter` | Start editing selected cell |
| `Tab` | Move to next cell |
| `Escape` | Cancel cell edit |
